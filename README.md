# Project Alpine

A task manager web app designed for balancing academics and athletics.

## Vision

A clean, intuitive task management system inspired by Notion and Obsidian, built for students who need to juggle coursework, assignments, practice schedules, and competitions.

## Core Features (Planned)

- **Task CRUD**: Create, read, update, delete tasks with ease
- **Categories**: Organize by class, sport, personal, etc.
- **Priority Levels**: High, medium, low with visual indicators
- **Due Dates**: Calendar integration and deadline tracking
- **Archive System**: Don't delete - archive completed or old tasks
- **Multiple Views**: List, Kanban board, Calendar
- **Tags & Filters**: Quick filtering and search
- **Dark/Light Mode**: Easy on the eyes during late-night study sessions
